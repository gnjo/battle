```
防具は特化させる。前衛、後衛で基本は考える。
武器：職種毎の補正。
流盾：物理ダメージを回避、前衛職のみ。
星衣：魔法ダメージをカット。
装飾：状態異常、生命力増強、スキル付与。
装飾：状態異常に関して。
装飾：状態異常に関して。

ランクは４段階。
ランク１＆２は序盤のドロップテーブル。
ランク２＆３は中盤のドロップテーブル。
ランク３＆４は終盤のドロップテーブル。

ランク１から４によって武具性能をかえる。
転職では魔法を引き継ぐ事は出来ない。
マジックユーザは自身の覚えている魔法を、装飾に込める事ができる。
魔法はレベルと同じだけ使える。前衛はレベルの半分。

```
```
主画面は依頼。依頼は常に一つのみ。依頼を達成すると自動帰還。
レベルは、達成帰還すると上がる。それまでは蓄積。未達成帰還では上がらない。
いつでも帰還できる。オートセーブされる。依頼を受けた段階で迷宮が生成される。受けている間は固定。

依頼は貢献金を払えば、上位の依頼を受ける事ができる。
依頼には、星の数があり、星の数の累積数でシナリオが進行する。次の章までの星は提示。
一度到達したレベルまでは１０倍の速さで上がる。最大値が継承される。
竜石が貰える。最大レベルを上限として生命力を加算する。
星の数を満たすと、討伐が発生し、ドロークとの討伐となる。

依頼：酒場で依頼を受ける。
商館：物品の売買。
壺屋：装飾に魔技を加える。
羊角：冒険者を見る。
迷宮：転移門を使い迷宮へ。
```
